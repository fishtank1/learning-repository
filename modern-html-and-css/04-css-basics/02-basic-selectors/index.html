HTML & CSS Sandbox - Basic Selectors

Complete list of selectors

  • Universal Selector (*)
  • Type Selector (Element Selector)
  • Class Selector (.classname)
  • ID Selector (#id)
  • Descendant Selector (ancestor descendant)
  • Child Selector (parent > child)
  • Adjacent Sibling Selector (A + B)
  • General Sibling Selector (A ~ B)
  • Attribute Selectors - Targets elements based on their attributes.
  • Grouping Selector (A, B)
  • Pseudo-classes - Targets elements in a specific state.
  • Pseudo-elements - Targets parts of an element.
  • Compound Selectors (A.B, A#B, etc.) ➡️ NEW
  • Combinators (Descendant, Child, Sibling)
      Define relationships between selectors:
    • Descendant Selector: A B
    • Child Selector: A > B
    • Adjacent Sibling Selector: A + B
    • General Sibling Selector: A ~ B
  • Negation (:not())
